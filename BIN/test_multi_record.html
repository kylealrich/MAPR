Multi-Record-Type Mapper Test

Test Instructions

  1. Open JavaScript_Mapper.html in your browser
  2. Select "Fixed-Length" file type
  3. Check the "Multi-Record-Type File" checkbox
  4. Upload the following files:
    • Input Data File: input/omrq.bcs.20250405101502.txt
    • Record Type Configuration: input/OMRQ_RecordType_Config.csv
  5. Click "Preview Files" - you'll be prompted to select the 4 mapping files:
    • input/OMRQ_Header_FixedLength_Mapping.csv
    • input/OMRQ_Line_FixedLength_Mapping.csv
    • input/OMRQ_Detail_FixedLength_Mapping.csv
    • input/OMRQ_Comment_FixedLength_Mapping.csv
  6. Review the preview showing records grouped by type (H/L/D/C)
  7. Click "Generate Mapper JS + Output CSV"
  8. Verify the output:
    • JavaScript Mapper: Contains processMultiRecordTypeFile() function
    • Hierarchical JSON: Shows parent-child relationships (Headers → LineItems → Details, Headers → Comments)
  9. Download both files and verify they're valid
